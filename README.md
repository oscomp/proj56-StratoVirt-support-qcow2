# proj56-StratoVirt-support-qcow2
### 项目名称
StratoVirt支持qcow2后端文件格式

### 项目描述

StratoVirt是openEuler的轻量虚拟化软件，基于Rust语言编写，具备功能完善、启动快速、内存开销小等特点。
目前支持的后端文件格式只有raw格式，为了更好的兼容性，拟支持qcow2文件格式。
